Projects

STM32F7 Screen Controller Board

Custom PCB designed around the STM32F7 microcontroller for an 8080 parallel interface screen controller application. Has an ESP32-C6 as a coprocessor for Wi-Fi connectivity and remote control capabilities. Serves as the main hub for an IOT smart home system, allowing users to control and monitor various smart devices through a user-friendly interface on the screen.

ESP32-C6 Environmental Sensor Node

Wireless environmental monitoring node based on the ESP32-C6 with Wi-Fi and Bluetooth connectivity. Serves as a slave device in a smart home system, collecting data from various sensors (temperature, humidity, air quality) and transmitting it to a central hub for processing and visualization. to feed into the STM32F7 screen controller board.

STM32WB65 Development Board

Development platform featuring the STM32WB65 wireless microcontroller for embedded and IoT development.
